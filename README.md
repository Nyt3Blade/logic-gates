Try out the project here:  https://logic-gates-phi.vercel.app/

Click on the playground button to play with the logic gates.

How to use the playground:
Drag and drop the items that you wish to use.
Connect the gates with values by clicking and dragging the blue circles next the object, they act as terminals for connections, you can right click these terminals to access which objects they are connected to and also remove those connections.
The bulb can be used to detect the output from gates
